# 🌿 Chili Leaf Disease Detection System

CNN-based web application for chili leaf disease detection built using Flask and TensorFlow. Users can upload images to detect diseases with confidence scores and receive causes, treatments, and per-acre recommendations. Includes multilingual support (English, Hindi, Telugu) and voice output.

---

## 🚀 Features

- 📸 Upload chili leaf image
- 🧠 Disease prediction using CNN (TensorFlow/Keras)
- 📊 Confidence score display
- 🌍 Multi-language support:
  - English 🇬🇧
  - Hindi 🇮🇳
  - Telugu 🇮🇳
- 🔊 Voice output (Text-to-Speech)
- 💊 Treatment suggestions
- 🌱 Quantity recommendation (per acre)
- 🔐 User authentication (Login / Signup / Forgot)

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, Bootstrap
- **Backend:** Flask (Python)
- **Model:** CNN (TensorFlow / Keras)
- **Database:** SQLite
- **Other:** NumPy, JavaScript (Speech API)

## ▶️ How to Run

### 1. Clone the repository

git clone https://github.com/yourusername/chili-leaf-disease-detection.git

cd chili-leaf-disease-detection

### 2. Install dependencies

pip install flask tensorflow numpy

### 3. Run the app

python app.py

### 4. Open in browser

http://127.0.0.1:5000
